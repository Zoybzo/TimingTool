# TimingTool
A timing tool implemented using Electorn.

## Start
You can start it by input `./HelloWorld {number} {choose}` in cmd.

The value of `{number}` is unnecessary whose existence is a mistake.
For {choose}, you can input in the following way.
|Value|InputParamsBy|
|---|---|
|0|cmd (the format is: hh mm ss)|
|1|window|
|2|files in the folder called config|

## config
config.json has the path about time.json for your time and text.json for your text.
